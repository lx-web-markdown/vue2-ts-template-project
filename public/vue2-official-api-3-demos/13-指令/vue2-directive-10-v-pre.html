Vue2 指令 v-pre 示例

介绍：

v-pre 指令用于跳过该元素及其所有子元素的编译过程。
主要用途是显示原始的 Mustache 标签 ({{ ... }})。
当一个节点包含大量不需要编译的内容时，使用 v-pre 也可以提升编译性能。

示例：使用 v-pre 显示原始模板语法

不使用 v-pre (会被编译)

{{ message }} 

        <div>{{ message }}</div>

使用 v-pre (跳过编译)

{{ message }} 
<div v-pre>{{ message }}</div>

更多说明

- v-pre 不需要表达式，它只是一个标记属性。
- 当你有很多静态内容，且其中没有 Vue 指令或插值时，可以考虑使用 < code>v-pre< /code> 来优化性能，尽管在大多数情况下这种优化是微不足道的。
