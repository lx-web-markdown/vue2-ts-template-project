Vue2 实例属性 vm.$isServer 示例

介绍：

vm.$isServer 是一个只读属性，用于判断当前 Vue 实例是否运行在服务器端（即服务端渲染 SSR）。
在浏览器环境下，该属性始终为 false。
只有在使用 Vue 服务端渲染 时，该属性才会为 true。 

// 判断当前是否为服务端渲染
console.log(vm.$isServer); // 浏览器环境下为 false

示例1：展示 vm.$isServer 的值

当前是否为服务端渲染：{{ isServer }}
// 访问实例属性
console.log(this.$isServer); // false（浏览器环境）

更多说明

- vm.$isServer 主要用于需要区分服务端和客户端渲染逻辑的场景。
- 在普通前端项目中，该属性通常为 false。
- 在 SSR（服务端渲染）环境下，该属性为 true，可用于条件渲染或避免访问浏览器专有 API。 

// SSR 场景下的用法示例
if (this.$isServer) {
  // 只在服务端执行的逻辑
} else {
  // 只在客户端执行的逻辑
}
