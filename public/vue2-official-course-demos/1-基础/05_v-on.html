v-on 事件绑定指令

  • v-on 指令用于监听 DOM 事件，并在触发时执行表达式或方法。
  • 常用语法：v-on:事件名="方法名"，也可简写为 @事件名="方法名"。
  • 点击下方按钮会调用 reverseMessage 方法，实现消息的反转。
<div id="app-5">
  <p>{{ message }}</p>
  <button v-on:click="reverseMessage">反转消息</button>
</div>

<script>
var app5 = new Vue({
  el: '#app-5',
  data: {
    message: 'Hello Vue.js!'
  },
  methods: {
    reverseMessage: function () {
      this.message = this.message.split('').reverse().join('');
    }
  }
})
</script>

{{ message }}
