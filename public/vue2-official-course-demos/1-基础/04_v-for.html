v-for 列表渲染指令

  • v-for 指令可以基于一个数组来渲染一个列表。
  • v-for 指令需要使用 item in items 形式的特殊语法。
  • 在控制台输入 app4.todos.push({ text: '新项目' })，你会发现列表最后添加了一个新项目。
<div id="app-4">
  <ol>
    <li v-for="todo in todos">{{ todo.text }}</li>
  </ol>
</div>

<script>
var app4 = new Vue({
  el: '#app-4',
  data: {
    todos: [
      { text: '学习 JavaScript' },
      { text: '学习 Vue' },
      { text: '整个牛项目' }
    ]
  }
})
</script>

待办事项列表：

  1. {{ todo.text }}
💡 提示：打开浏览器控制台，输入 app4.todos.push({ text: '新项目' }) 来添加新项目！
