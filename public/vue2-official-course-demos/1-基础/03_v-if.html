v-if 条件渲染指令

  • v-if 指令用于根据表达式的真假条件，动态地在 DOM 中插入或移除元素。
  • 如果 seen 为 true，则显示内容；否则不渲染该元素。
  • 你可以点击下方按钮切换 seen 的值，体验 v-if 的效果。
<div id="app-3">
  <p v-if="seen">现在你看到我了</p>
  <button @click="seen = !seen">切换显示/隐藏</button>
</div>

<script>
var app3 = new Vue({
  el: '#app-3',
  data: {
    seen: true
  }
})
</script>
