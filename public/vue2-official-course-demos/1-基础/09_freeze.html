Object.freeze 与 Vue 响应式

  • Object.freeze(obj) 可以冻结一个对象，使其属性不可变。
  • Vue 实例的 data 如果是被冻结的对象，响应式系统将无法追踪其变化。
  • 此时，视图不会随着数据的修改而更新。
var obj = { foo: 'bar' }

Object.freeze(obj)

new Vue({
  el: '#app-freeze',
  data: obj
})

{{ foo }}

说明：点击按钮后，foo 的值不会发生变化，视图也不会更新，因为 obj 已被冻结，Vue 无法追踪其变化。
