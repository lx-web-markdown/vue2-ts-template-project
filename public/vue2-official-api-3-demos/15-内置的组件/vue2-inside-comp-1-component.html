Vue2 内置组件 <component> 示例

介绍：

<component> 是 Vue 的内置组件，用于动态地绑定并渲染已注册的组件类型。
通过 is 属性，可以根据数据动态切换不同的组件，非常适合做动态组件、Tab 切换等场景。 

<component :is="currentComp"></component>

示例：动态切换组件

// 注册两个组件
Vue.component('user-card', {
  template: `< div class="data-box">👤 用户：小明   年龄：18< /div>`
});
Vue.component('product-card', {
  template: `< div class="data-box">📦 产品：Vue.js   版本：2.x< /div>`
});
// 动态切换
new Vue({
  el: '#component-demo',
  data: {
    currentComp: 'user-card'
  }
});

更多说明

- <component :is="..."> 可以绑定字符串（组件名）或组件对象。
- 适合做动态内容、Tab 切换、弹窗等场景。
- 也可与 keep-alive 结合缓存组件状态。
