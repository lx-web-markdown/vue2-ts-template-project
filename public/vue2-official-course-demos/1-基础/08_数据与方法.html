Vue 实例 - 数据与方法

  • 当一个 Vue 实例被创建时，它将 data 对象中的所有 property 加入到 Vue 的响应式系统中。
  • 当这些 property 的值发生改变时，视图将会产生"响应"，即匹配更新为新的值。
  • Vue 实例还暴露了一些有用的实例 property 与方法，它们都有前缀 $。

1. 基础数据绑定

// 数据对象
var myData = { a: 1 }

// Vue 实例
var vm = new Vue({
  el: '#app-7',
  data: myData
})

// 数据是响应式的
vm.a === myData.a // => true
vm.a = 2
myData.a // => 2
当前 a 的值: {{ a }}

2. 响应式系统演示

// 通过原始数据对象修改
myData.a = 100

// 通过 Vue 实例修改
vm.a = 200

// 两种方式都会触发视图更新
原始数据对象 myData.a: {{ originalDataA }}
Vue 实例 vm.a: {{ a }}

3. Vue 实例属性 ($ 前缀)

// 实例属性都有 $ 前缀
vm.$data === myData // => true
vm.$el === document.getElementById('app-7') // => true

// 实例方法
vm.$watch('a', function (newValue, oldValue) {
  console.log('a 从', oldValue, '变为', newValue)
})
$data 对象: {{ JSON.stringify($data) }}
$el 元素: {{ $el ? $el.tagName : 'null' }}
监听器数量: {{ watcherCount }}

4. 非响应式属性

// 只有创建时存在的 property 才是响应式的
vm.b = 'hi' // 这个改动不会触发视图更新

// 如果知道晚些时候需要某个 property，应该预先设置初始值
data: {
  newTodoText: '',
  visitCount: 0,
  hideCompletedTodos: false,
  todos: [],
  error: null
}
响应式属性 a: {{ a }}
非响应式属性 b: {{ b || '未定义' }}

5. Object.freeze() 演示

var obj = { foo: 'bar' }
Object.freeze(obj)

new Vue({
  el: '#app',
  data: obj
})
