v-model 双向数据绑定

  • v-model 指令用于在表单控件元素和 Vue 实例的数据之间实现双向数据绑定。
  • 输入框的内容会实时同步到 message，反之亦然。
  • 你可以在下方输入框中输入内容，体验 v-model 的效果。
<div id="app-6">
  <p>{{ message }}</p>
  <input v-model="message" />
</div>

<script>
var app6 = new Vue({
  el: '#app-6',
  data: {
    message: 'Hello Vue!'
  }
})
</script>

{{ message }}
