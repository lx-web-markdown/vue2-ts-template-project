Vue2 实例属性 vm.$el 示例

介绍：

vm.$el 是 Vue 实例的一个属性，指向实例所挂载的 DOM 元素。
你可以通过 this.$el 访问到 Vue 实例对应的根 DOM 节点。
这个属性在实例挂载后（即 mounted 钩子之后）可用。 

// 访问实例根 DOM 元素
console.log(vm.$el);

详细说明请参考 Vue2 官方文档 - vm.$el

示例1：展示 vm.$el 内容

Hello Vue2！
// HTML 结构
<div id="my-el">Hello Vue2！</div>

// 访问实例根 DOM 元素
console.log(this.$el); // 输出对应的 DOM 节点

示例2：修改 vm.$el 的样式

// 通过 this.$el 修改 DOM 样式
this.$el.style.background = '#fffbe6';
this.$el.style.border = '2px solid #faad14';

更多说明

- vm.$el 只读，指向 Vue 实例挂载的根 DOM 元素。
- 通常用于需要直接操作 DOM 的场景。
- 推荐优先使用 Vue 的数据驱动方式，只有在必须操作 DOM 时才用 vm.$el。
