Vue2 指令 v-once 示例

介绍：

v-once 指令用于执行一次性地插值，当数据改变时，插值处的内​​容不会更新。
这可以用于优化更新性能，特别是当您需要渲染大量静态内容时。 

<!-- 单个元素 -->
<span v-once>这个值不会被改变: {{ msg }}</span>

<!-- 带有子元素的元素 -->
<div v-once>
  <h1>评论</h1>
  <p>{{ msg }}</p>
</div>

示例：v-once 的使用

会更新的消息: {{ message }}
不会更新的消息 (v-once): {{ message }}

<div id="app">
  <div>会更新的消息: {{ message }}</div>
  <div v-once>不会更新的消息 (v-once): {{ message }}</div>
  <button @click="updateMessage">更新消息</button>
</div>
<script>
  var vm = new Vue({
    el: '#app',
    data: {
      message: '初始消息'
    },
    methods: {
      updateMessage() {
        this.message = '已更新消息 - ' + new Date().toLocaleTimeString();
      }
    }
  });
</script>
