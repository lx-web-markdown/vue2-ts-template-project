Vue2 实例属性 vm.$options 示例

介绍：

vm.$options 是 Vue 实例的一个属性，包含实例初始化时传递的所有选项对象。
你可以通过 this.$options 访问到 data、methods、props、computed、watch 等所有配置。
这对于调试、插件开发或高级用法非常有用。 

// 访问实例选项对象
console.log(vm.$options);

示例1：展示 vm.$options 的部分内容

标题：{{ title }}
作者：{{ author }}
// 选项对象
data: {
  title: 'Vue2 $options 示例',
  author: '小明'
},
methods: {
  showOptions() {
    console.log(this.$options);
  }
}
// 访问实例选项对象
console.log(this.$options);

示例2：访问自定义选项

// 访问自定义选项
this.$options.myCustomOption

更多说明

- vm.$options 包含实例初始化时的所有选项。
- 可以通过 this.$options.data、this.$options.methods 等访问原始配置。
- 也可以添加自定义选项，如 myCustomOption，通过 this.$options.myCustomOption 访问。
- 适合调试、插件开发等高级场景。
