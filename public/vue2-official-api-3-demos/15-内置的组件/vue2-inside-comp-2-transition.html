Vue2 内置组件 <transition> 示例

介绍：

<transition> 是 Vue2 内置的过渡动画组件。
它为进入/离开 DOM 的元素或组件提供过渡效果。
只需包裹需要过渡的元素，并为其设置 CSS 过渡类名即可。 

<transition name="fade">
  <div v-if="show">内容</div>
</transition>

示例1：淡入淡出动画

Hello，Vue2 transition！ 
<transition name="fade">
  <div v-if="show">Hello，Vue2 transition！</div>
</transition>

data: {
  show: true
},
methods: {
  toggle() {
    this.show = !this.show;
  }
}

更多说明

- <transition> 只渲染一个子元素或组件。
- 支持多种过渡方式：CSS 过渡、CSS 动画、JavaScript 钩子。
- 常用属性：name、appear、mode 等。
- 详细用法见 官方文档。
