Vue2 指令 v-cloak 示例

介绍：

v-cloak 指令用于防止页面加载时未编译的 Mustache 标签（如 {{ message }}）闪现。
通常配合 CSS [v-cloak] { display: none; } 使用，直到 Vue 实例编译完成后自动移除 v-cloak 属性。 

<div v-cloak>{{ message }}</div>
<style>
  [v-cloak] { display: none; }
</style>

示例：防止 Mustache 标签闪现

{{ message }} 
<div id="app">
  <div v-cloak>
    {{ message }}
  </div>
</div>

<script>
  new Vue({
    el: '#app',
    data: {
      message: 'Hello v-cloak!'
    }
  });
</script>

- 当 Vue 未加载完成时，v-cloak 属性存在，内容被隐藏。
- Vue 加载并编译后，v-cloak 属性被移除，内容显示。
